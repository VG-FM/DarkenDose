# DarkenDose

DarkenDose is a MATLAB-based tool for radiochromic film dosimetry.

## Features

* Optical density calculation
* Dose reconstruction (cGy)
* Dose maps and isodose curves
* ROI-based analysis

## How to use

1. Run the script in MATLAB
2. Select irradiated film (TIFF)
3. Select reference film (0 cGy)
4. Define ROI
5. Obtain dose maps and statistics

## Requirements

* MATLAB R2023b or newer
